# Website Daftar Utang 3D Interaktif

Website untuk mengelola daftar utang dengan visualisasi 3D dan sistem CRUD lengkap.

## Fitur
- Visualisasi 3D dengan Three.js
- Sistem CRUD (Create, Read, Update, Delete)
- Responsif untuk semua device
- Animasi dan interaksi menarik

## Teknologi
- HTML5, CSS3, JavaScript
- Three.js untuk 3D
- Vercel untuk hosting
